### 📝 Personal Reflection
I really enjoyed building this project for my CSC 202 course at **MAAUN**. It helped me understand how to bridge the gap between terminal logic and a real web application. It was a great challenge to implement version control while building functional code.

### 🚀 Project Goal
The goal of this directory is to help local businesses in **Kano, Nigeria**, gain visibility through a simple, efficient web interface.

### 🛠️ Features
* **OOP-based Business models** (using Python Classes)
* **Stack Data Structure** (LIFO logic for "Recently Added" history)
* **Flask Web Routing** (handling backend navigation)
* **HTML Templates** (clean user interface for the directory)

### 💻 How to Run This Project
1. **Install Flask**: `pip install Flask`
2. **Run the app**: `python app.py`
3. **Visit**: `http://127.0.0.1:5000` in your browser.
### 🛠️ Tech Stack: Python, Flask, HTML5, CSS3, Git
#### 📚 Data Structures: Used a Python List as a Stack (LIFO) for history.
